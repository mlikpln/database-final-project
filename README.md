# Итоговый проект по SQL

Решение практических задач по анализу данных и написанию различных SQL-запросов

## Цели и задачи проекта
* Закрепление навыков работы с реляционными базами данных
* Написание аналитических запросов с использованием группировок, подзапросов, объединений и рекурсивных CTE
* Оптимизация структуры запросов для эффективной выборки данных

## Стек технологий
* **СУБД:** MySQL 8.0
* **Инструменты:** phpMyAdmin

## Структура баз данных и реализованные функции
Проект включает в себя работу с четырьмя базами данных:
1. **Vehicle:** Была сделана выборка транспорта по техническим характеристикам и объеденение различных данных в общую таблицу
2. **Racing:** Был реализован анализ результатов гонок, написаны запросы для поиска самых лучших машин в своих классах, расчет средних позиций, фильтрация по эффективности, вычисление метрик классов с помощью CTE
3. **Hotel booking:** Были написаны  запросы для выгрузки статистики клиентов и классификации клиентов по их расходам через конструции 'CASE'
4. **Company:** Реализовано построение сложных организационных структур в компании, при  помощи рекурсивных запросов выведена цепочка подчиненных, подсчет прямых и не прямых подчиненныхз для менеджеров и вывод списков задач и проектов по конкретным  отделам

## Структура репозитория
Каждая база данных вынесена в отдельную папку. Внутри папки находятся два файла:
1. **'init.sql'** - скрипт создания таблиц для MySQL и наполнения их тестовыми данными
2. **'solutions.sql'** - файлы с готовыми решениями задач

## Папки проектов:
* '01_vehicle' 
* '02_racing'
* '03_hotel_booking'
* '04_company'

## Инструкция по запуску и тестированию
1. Откройте нужную папку 
2. Запустите скрипт 'init.sql' в вашей СУБД для создания таблиц и наполнения их данными
3. Откройте файл 'solutions.sql', скопируйте нужный запрос и запустите его для проверки работы
